# How to Use This Card Deck

## Card Format
Each card includes:
- **Card Type & Points**: Easy/Medium/Hard Prompt OR Debate
- **Scenario/Proposition**: The business situation or debate topic
- **Challenge Instructions**: What the team must do
- **Time Limit**: How long teams have
- **Instructor Notes**: Scoring guidance available in separate instructor-notes-v2.md file

## Revealing Cards
Options:
1. **Project on screen**: Display card, read aloud
2. **Print and distribute**: Hand to accepting team
3. **Digital handout**: Share via Google Doc (paste card into a table with light orange background RGB 255, 229, 153)

## Suggested Order
Cards are numbered but can be used flexibly. Consider:
- Start each session with easy warm-ups
- Vary between prompts and debates
- Save hardest cards for when teams are warmed up
- Use easy cards for lightning round catch-up
